# Product

## Register

product

## Users

People using EweserDB and EweNote to keep local-first notes, project memory,
links, metadata, and optional sync under their own control.

## Product Purpose

EweNote is a calm local-first note workspace: fast writing first, Obsidian-style
capability nearby, and metadata available only when the user asks for it.

## Brand Personality

Sparse, private, capable.

## Anti-references

Avoid SaaS dashboards, busy command centers, labeled toolbar clutter, low
contrast novelty minimalism, and note apps where UI text competes with the
user's writing.

## Design Principles

- The user's note text is the interface.
- Hide capability until intent is clear.
- Panels are spatial memory, not marketing surfaces.
- Local-first trust beats decorative polish.
- Metadata belongs in the inspect panel, not in the writing canvas.

## Accessibility & Inclusion

Target readable contrast in dark and light themes, keyboard reachable panel
controls, visible focus states, and reduced-motion-safe transitions.
