## Manual QA notes

- `.canvas` and `.base` files should open in Obsidian without obvious syntax
  errors.
- Attachment inventory is intentionally tiny. Some non-image media files are
  placeholder assets meant for file-preservation coverage rather than playback
  quality.
- Duplicate names and alias collisions are deliberate. Do not "clean them up."
