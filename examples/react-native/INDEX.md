# @eweser/example-react-native

## Plain English

This example demonstrates EweserDB usage in a React Native app.

## Owns

- Mobile example source, status/login components, and React Native app
  configuration.

## Start Here

- [`package.json`](./package.json): React Native/Expo scripts.
- [`src/INDEX.md`](./src/): Source navigation map.
- [`src/AppReactNative.tsx`](./src/AppReactNative.tsx): Main mobile app
  component.

## Children

- [`assets/`](./assets/): React Native assets.
- [`src/`](./src/): React Native app source.

## Key Contracts

- Mobile-specific storage/network assumptions should stay isolated to this
  example and SDK-supported configuration.

## Update Triggers

- Update when mobile app structure, SDK usage, assets, or scripts change.

## Testing

- `npm run web --workspace @eweser/example-react-native`: Starts the Expo web
  target when needed.
