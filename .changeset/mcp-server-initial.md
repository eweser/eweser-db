Initial release of the EweserDB MCP server (`@eweser/mcp`).

Provides a Model Context Protocol (stdio) server for AI agents to read and write EweserDB rooms via agent tokens. Includes tools: `eweser_list_rooms`, `eweser_list_documents`, `eweser_get_document`, `eweser_search`, `eweser_create_document`, `eweser_update_document`, `eweser_delete_document`, `eweser_save_memory`.
